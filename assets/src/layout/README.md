# Layout Folder
This folder is reserved top level site structure. Logos and other branding
elements may also go in this folder.

## SCSS Files
All `*.scss` files must begin with an underscore, except the `layout.scss`
"manifest" file.

## Storybook Files
Not every component needs a storybook file.

## Webpack Entry Point Files
`layout.js` is the single entry point for this folder. No subfolders will have
entry point `*.js` files.
